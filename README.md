# Spotify Web Player Clone

A pixel-accurate clone of Spotify's web player UI built with HTML and CSS.

## Features
- Sidebar with navigation and library section
- Scrollable main content with music cards
- Fixed music player at the bottom with playback controls
- Responsive layout using CSS Flexbox
- Mobile adaptability via media queries
- Spotify dark theme design system

## Tech Stack
HTML | CSS
